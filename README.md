## ğŸš€ Deployment

### GitHub Pages

1. Create a new repository on GitHub
2. Push your code:

```bash
git init
git add .
git commit -m "Initial commit: Portfolio website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git push -u origin main
```

3. Go to repository Settings â†’ Pages
4. Select `main` branch as source
5. Your site will be live at `https://YOUR_USERNAME.github.io/portfolio`

### Hugging Face Spaces

1. Create a new Space on Hugging Face
2. Select "Gradio" as SDK
3. Clone the space repository:

```bash
git clone https://huggingface.co/spaces/YOUR_USERNAME/portfolio
```

4. Copy all files to the cloned directory
5. Push to Hugging Face:

```bash
git add .
git commit -m "Add portfolio website"
git push
```

Your portfolio will be live at `https://huggingface.co/spaces/YOUR_USERNAME/portfolio`
